## About The Project

Ever want to convert an image into an ASCII-generated image? This small web application will do just that. Simply upload your own image, then convert for in-browser viewing or download. The downloaded files are .PNG not a .txt file like you might expect.

## Getting Started

The front and backend need to be started seperatly, currently the Flask backend is stored inside the /api folder.

### Installation

1. Clone Repo

```sh
  git clone https://github.com/natespring/ASCII-Image-Converter.git
```

2. Install NPM packages
   ```sh
   npm install
   ```
3. Start Flask server
   ```js
   cd api && python main.py
   ```
4. Start React frontend
   ```js
   npm start
   ```
